## Install
    $ npm install

## Configure app

Edit `./app.log` then update it with your logs.

## Lint the project

    $ npm run lint

## Format the project with prettier

    $ npm run format
## Test the project with Jest

    $ npm run test
## Build the project

    $ npm run build

## Run the project

    $ node parser.js --input ./app.log --output ./errors.json

## Output will store
`./errors.json` as  expected output.
